Unix Shell Script

This project spanned over the duration of the whole semester. We began with a simple program to parse user input and just seperate and detect each word. Over a few iteration we built a shell script to handle specific tasks . This behaves like a command line interpreter for the unix system we used. As such it would print a prompt ":480:" in this case wait for intructions like: multiple pipline intructions, ls-f, exec, metacharacters, and more. The important part for this was properly forking a child process to handle the spcific instruction while keeping the main process running and waiting. My skills with C vastly improved as this was the most complicated work I have done with C.
